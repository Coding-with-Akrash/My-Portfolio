Jarvis Voice Assistant v1.5.3

System Requirements

  • Windows 10/11 or macOS
  • Python 3.8+
  • 4GB RAM minimum
  • Working microphone

Dependencies

  • speech_recognition
  • pyttsx3
  • wikipedia
  • pywhatkit

What It Does

Your desi AI assistant that understands Pakistani English and responds like a local helper. Perfect for:

  • Quick information lookup
  • Hands-free web searching
  • Playing your favorite songs
  • Getting answers without typing

How to Install

  1. Make sure Python is installed: python --version
  2. Download the Jarvis files
  3. Run: python jarvis.py
  4. It will auto-install required packages

How to Use

After starting, choose your preferred mode:

1. Type Mode - For quiet environments
2. Voice Mode - For hands-free operation (recommended)

Common Commands

General Knowledge:
"Jarvis, who is Allama Iqbal?"
"Jarvis, what is the capital of Pakistan?"
Web Searches:
"Jarvis, search in google for today's dollar rate"
"Jarvis, search in youtube for Naseebo Lal songs"
Entertainment:
"Jarvis, music on youtube Atif Aslam"
"Jarvis, music on youtube Bollywood hits"

What's New in v1.5.3

  • Better Urdu/English mixed speech recognition
  • Faster response time for Pakistani accents
  • New voice profiles with desi tone
  • Auto-install of missing packages

Troubleshooting

  • Mic not working? Check system permissions
  • Slow responses? Check internet speed
  • Installation errors? Try: pip install --upgrade pip
