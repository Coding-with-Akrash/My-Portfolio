Comparative Analysis of Deep Learning Models for Chest X-Ray Image Classification

By Akrash Noor | December 21, 2025 | AI, Machine Learning, Medical Imaging

Abstract

Chest X-ray imaging is one of the most widely used diagnostic tools for detecting pulmonary diseases. Recent advances in deep learning, particularly Convolutional Neural Networks (CNNs), have significantly improved automated medical image analysis. This project presents a comparative evaluation of multiple deep learning models trained on chest X-ray images using PyTorch and TensorFlow frameworks. The study includes mathematical modeling of CNN operations, optimization techniques, and loss functions to provide a complete theoretical and practical understanding of model performance.

1. Introduction

Medical image classification can be formulated as a supervised learning problem, where the goal is to learn a function:

fθ: X → Y

where:

  • X represents the space of chest X-ray images,
  • Y denotes disease labels (e.g., Normal, Pneumonia),
  • θ are trainable model parameters.

Given a dataset {(xi, yi)i=1N}, the objective is to learn θ such that the predicted label ŷi = fθ(xi) closely matches the ground truth yi.

2. Dataset Representation and Preprocessing

Each chest X-ray image is represented as a tensor:

X ∈ ℝH×W×C

where:

  • H = image height,
  • W = image width,
  • C = number of channels (typically C=1 for grayscale).

Normalization

Pixel values are normalized to improve training stability:

Xnorm = (X - μ) / σ

where μ and σ are the dataset mean and standard deviation.

3. Convolutional Neural Network (CNN) Architecture

3.1 Convolution Operation

The convolution layer extracts spatial features using kernels (filters):

Zi,j,k = ∑m=0M-1 ∑n=0N-1 Xi+m,j+n ⋅ Km,n,k + bk

where:

  • K is the convolution kernel,
  • bk is the bias term,
  • k is the feature map index.

3.2 Activation Function (ReLU)

To introduce non-linearity, the Rectified Linear Unit (ReLU) is applied:

f(x) = max(0, x)

This helps CNNs learn complex patterns in chest X-ray images.

3.3 Pooling Layer

Pooling reduces spatial dimensions while preserving important features:

Pi,j = max(m,n)∈Ω Zi+m,j+n

Max-pooling improves translation invariance and reduces computation.

4. Fully Connected Layers

After feature extraction, flattened feature maps are passed to dense layers:

h = σ(Wx + b)

where:

  • W is the weight matrix,
  • b is the bias vector,
  • σ is an activation function.

5. Output Layer and Softmax Function

For classification, the Softmax function converts logits into probabilities:

P(y=k|x) = ezk / ∑j=1C ezj

where:

  • zk is the logit for class k,
  • C is the number of classes.

6. Loss Function: Cross-Entropy Loss

The objective function minimized during training is the categorical cross-entropy loss:

L = -∑i=1C yi log(ŷi)

where:

  • yi is the true label,
  • ŷi is the predicted probability.

7. Optimization Using Gradient Descent

Model parameters are updated using gradient-based optimization:

θt+1 = θt - η ∇θ L

where:

  • η is the learning rate,
  • ∇θ L is the gradient of the loss.

Adam Optimizer

Adam adapts learning rates using moment estimates:

mt = β₁ mt-1 + (1-β₁) gt

vt = β₂ vt-1 + (1-β₂) gt2

θt = θt-1 - η √(vt + ε) / mt

8. Model Evaluation Metrics

Accuracy

Accuracy = (TP + TN) / (TP + TN + FP + FN)

Precision

Precision = TP / (TP + FP)

Recall

Recall = TP / (TP + FN)

F1-Score

F1 = 2 ⋅ (Precision ⋅ Recall) / (Precision + Recall)

These metrics are crucial for medical diagnosis where false negatives can be critical.

9. Model Comparison Strategy

All models were trained under identical conditions:

M = {M₁, M₂, …, Mₙ}

Each model Mi was evaluated using the same dataset, loss function, and metrics to ensure fairness and reproducibility.

10. Results and Discussion

Experimental results show that deeper CNNs and transfer learning models achieve higher classification accuracy due to richer feature representations. However, increased depth also raises computational cost. The trade-off between accuracy and efficiency is a key consideration in real-world clinical deployment.

11. Conclusion

This project demonstrates a mathematically grounded and experimentally validated comparison of deep learning models for chest X-ray classification. By integrating theoretical foundations with practical implementation in PyTorch and TensorFlow, the project highlights best practices in medical AI development.
